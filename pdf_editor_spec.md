# PDF編集Webアプリ 仕様書

## 概要

| 項目 | 内容 |
|------|------|
| アプリ名 | PDF Editor Local |
| 形態 | シングルHTMLファイル（ブラウザで動作） |
| 動作環境 | 完全ローカル・オフライン（クラウド不使用） |
| 対象OS | Windows / Mac / Linux（モダンブラウザ共通） |
| ライセンス | 個人利用自由 |

---

## 技術スタック

| 役割 | 技術 |
|------|------|
| UI | HTML / CSS / JavaScript（バニラ） |
| PDF操作 | [PDF-lib](https://pdf-lib.js.org/)（ブラウザ内で動作するJSライブラリ） |
| PDFレンダリング（プレビュー） | [PDF.js](https://mozilla.github.io/pdf.js/)（Mozilla製） |
| ファイル保存 | FileSaver.js または Blob URL |
| 外部通信 | **なし**（完全オフライン動作） |

---

## 機能一覧

### 1. ファイル読み込み
- ドラッグ＆ドロップでPDFを読み込む
- ファイル選択ダイアログからも読み込み可能
- 複数ファイルの同時読み込み（結合用）
- 読み込み後、全ページをサムネイル表示

### 2. ページプレビュー
- 全ページをサムネイル（グリッド）表示
- サムネイルクリックで拡大プレビュー
- ページ番号表示
- 選択中ページのハイライト表示

### 3. ページ削除
- サムネイルをクリックして選択（複数選択可）
- 「削除」ボタンで選択ページを削除
- 削除前に確認ダイアログ表示

### 4. ページ並び替え
- サムネイルをドラッグ＆ドロップで順序変更
- 変更はリアルタイムでサムネイルに反映

### 5. 回転
- 選択ページを90°単位で回転（右回転・左回転）
- 全ページ一括回転オプション
- サムネイルにリアルタイム反映

### 6. 結合
- 複数PDFファイルを読み込み
- ドラッグ＆ドロップでファイル順序を変更
- 指定順序で1つのPDFに結合して出力

### 7. 分割
- 指定ページで分割（例：1-3ページ、4-6ページ）
- 1ページずつ個別ファイルに分割
- 分割後は各ファイルを個別にダウンロード

### 8. 圧縮
- 画像品質を調整してファイルサイズを削減
- 圧縮レベルを選択（低・中・高）
- 圧縮前後のファイルサイズを表示

### 9. 出力・保存
- 「ダウンロード」ボタンで編集済みPDFを保存
- デフォルトファイル名：`元ファイル名_edited.pdf`
- ブラウザの標準ダウンロード機能を使用

## 操作フロー

### 基本フロー
1. アプリをブラウザで開く（HTMLファイルをダブルクリック）
2. PDFファイルをドラッグ＆ドロップ or ファイル選択
3. サムネイルで内容を確認
4. 編集操作（削除・回転・並び替えなど）
5. 「ダウンロード」で保存

### 結合フロー
1. 複数PDFをドラッグ＆ドロップ
2. ファイルリストで順序を調整
3. 「結合」ボタンをクリック
4. ダウンロード

### 分割フロー
1. PDFを読み込む
2. 「分割」ボタンをクリック
3. 分割方法を選択（ページ指定 or 全ページ個別）
4. 各ファイルをダウンロード

## 非機能要件

| 項目 | 要件 |
|------|------|
| プライバシー | ファイルはブラウザ内のみで処理。外部送信なし |
| オフライン動作 | CDNライブラリをHTMLに内包し、完全オフラインで動作 |
| 対応ブラウザ | Chrome / Edge / Firefox 最新版 |
| ファイルサイズ制限 | ブラウザのメモリに依存（目安：1ファイル100MB以内） |
| インストール | 不要（HTMLファイルをダブルクリックで起動） |

---

## 制約・注意事項

- パスワード保護されたPDFは非対応（解除機能なし）
- 圧縮はテキストPDFには効果が薄い（画像埋め込みPDFに有効）
- 大容量ファイル（100MB超）は処理が遅くなる場合がある
- IE（Internet Explorer）は非対応
